# 🚀 Online Bidding System

A scalable full-stack MERN application that enables real-time online bidding with secure authentication, dynamic product management, and a responsive UI.

---

## 🧠 Features

- 🔐 User Authentication (Login/Register)
- 📦 Product Listing & Management
- 💰 Real-time Bidding System
- 📊 Dynamic UI Updates
- 📱 Responsive Design

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Framer Motion

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
